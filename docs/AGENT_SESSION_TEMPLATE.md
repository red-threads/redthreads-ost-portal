# Agent Session Template

Use this for Baton pass, QA reviewer, and implementation handoffs.

## Mode

Architect, Debug, Produce final code, Baton pass, QA reviewer, Full ship, or Plan.

## Goal

State the task in one or two sentences.

## Current Branch And Status

Paste:

```bash
git status --short --branch
git log --oneline -5
```

## Files In Scope

List exact files that may be changed.

## Files Out Of Scope

List exact files that must remain untouched.

## Repo-Verified Facts

Summarize what was verified from files or commands.

## Prompt/Historical Facts

Summarize context that came from project memory but is not currently repo-verified.

## Validation

List commands run and results.

## Risks And Follow-Ups

List contradictions, blocked verification, and deferred work.
