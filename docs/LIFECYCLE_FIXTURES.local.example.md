# Lifecycle Fixtures Local Template

Copy this file to `docs/LIFECYCLE_FIXTURES.local.md` for private local use. Do not commit the local file.

## Fixture Access

Store only local/private fixture access notes here:

- Fixture number
- Local token or URL
- Expected lifecycle state
- Test notes

Never commit live tokens, tokenized URLs, raw snapshots, customer data, emails, org names, invoice numbers, PO numbers, or raw sheet rows.

## Accepted Fixture Family

- 1900 fresh estimate / reset
- 1916 quantities entered / artwork pending
- 1922 ready to order
- 1923 PO initiated / awaiting submission
- 1925 PO submitted unpaid / production active
- 2003 PO submitted unpaid / jobs complete
- 2004 PO submitted paid / print incomplete
- 2005 PO submitted paid / complete
- 2006 manual payment pending
- 2007 manual payment received / print incomplete
- 2008 manual payment received / complete
